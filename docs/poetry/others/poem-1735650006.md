# 登高

登高是一首唐代诗人杜甫创作的诗歌。

<details>
<summary>创作背景</summary>

这首诗作于唐代安史之乱后，是杜甫晚年漂泊到夔州时所作。诗人在重阳节登高望远，抒发了对时局的忧虑和对人生的感悟。

</details>

## 注释

<details>

- 渚清：水中小洲清晰可见
- 萧萧：形容落叶纷纷飘下的样子
- 滚滚：形容江水奔流的样子
