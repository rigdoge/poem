# 静夜思

静夜思是一首唐诗歌，作者是李白。

<details>
<summary>创作背景</summary>

这首诗作于唐代，具体创作年代已不可考。诗人在一个月明之夜，思念远方的家乡，抒发了思乡之情。

</details>

## 注释

<details>
<summary>词语注释</summary>

- 床前明月光：月光照在床前，形容夜深人静的景象
- 地上霜：地上的霜，这里是误以为月光是地上的霜
- 举头望明月：抬头看明月
- 低头思故乡：低头想念家乡

</details>
